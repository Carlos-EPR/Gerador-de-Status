# Gerador de Status para Itens em Jogos de RPG

Este é um programa em Python que gera aleatoriamente status para itens em jogos de RPG, atribuindo-lhes raridades com base nos valores gerados.

## Funcionalidades

### 1. Geração de Status
   - Gera aleatoriamente um valor de status para um item, com base em uma distribuição de probabilidades.
   - A probabilidade de gerar determinados valores de status é influenciada por um valor de sorte (lucky) gerado aleatoriamente.

### 2. Atribuição de Raridade
   - Atribui uma raridade ao item com base no valor de status gerado.
   - As raridades são classificadas em sete categorias: Bronze, Silver, Gold, Emerald, Diamond, Ruby, Obsidian e Opal.

## Utilização
1. Execute o arquivo `main.py`.
2. O programa irá gerar um valor de status e atribuir uma raridade ao item correspondente.
3. O resultado será exibido no console, indicando a raridade e o valor de status do item gerado.

## Pré-requisitos
- Python 3.x instalado no sistema.

## Execução
1. Abra um terminal.
2. Navegue até o diretório onde os arquivos do programa estão localizados.
3. Execute o seguinte comando:
   ```
   python main.py
   ```
